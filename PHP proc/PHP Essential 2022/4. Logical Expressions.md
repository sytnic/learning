## 024-Else and elseif statements

```php
<body>
    <?php
        $a = 3;
        $b = 4;
        
        if ($a > $b) {
            echo "a is larger than b";
        } elseif ($a < $b) {
            echo "a is smaller than b";
        } else {
            echo "a is equal to b";
        }			
    ?>		

</body>
```

```php
    $a = 4;

    }

    $a = 4;

    $c = 1;

        echo "c is larger than d";
    }			
?>

<br />
<?php
    $e = 100;
    if (!isset($e)) {
        $e = 200;
```
