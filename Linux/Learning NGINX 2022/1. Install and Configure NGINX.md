## 006-Install NGINX on Ubuntu

Все последующие команды от root:

    sudo su -

Обновление пакетов

    apt update

Установка

    apt install nginx

Проверка статуса, разные варианты

    nginx -v
    systemctl status nginx

По умолчанию веб-страница nginx, доступная по адресу 127.0.0.1, располагается в папке /usr/share/nginx/html/index.html.
