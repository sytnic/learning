## 029-Mix text and variables with template literals

Литерал шаблона - это контент для константы, заключенный в апострофы, 

```js
    ` //... `
```

могущий содержать HTML или смешанный HTML и JS.  

Например, вот `script.js`, который выведет HTML и значения JS на экран:

```js
import Backpack from "./Backpack.js";

const everydayPack = new Backpack(
  "Everyday Pack",
  30,
  "grey",
  15,
  26,
  26,
  false,
  "December 5, 2018 15:00:00 PST"
);

const content = `
  <main>
    <article>
      <h1>${everydayPack.name}</h1>
      <ul>
        <li>Volume: ${everydayPack.volume}</li>
        <li>Color: ${everydayPack.color}</li>
        <li>Age: ${everydayPack.backpackAge()}</li>
        <li>Number of pockets: ${everydayPack.pocketNum}</li>
        <li>Left strap length: ${everydayPack.strapLength.left}</li>
        <li>Right strap length: ${everydayPack.strapLength.right}</li>
        <li>Lid status: ${everydayPack.lidOpen}</li>
      </ul>
    </article>
  </main>
`;

document.body.innerHTML = content;
```

## 030-Traditional string output

Традиционный способ использовал конкатенацию и был не очень хорош при учитывании пробелов.  

Пример, `script.js`

```js
import Backpack from "./Backpack.js";

const everydayPack = new Backpack(
  "Everyday Pack",
  30,
  "grey",
  15,
  26,
  26,
  false,
  "December 5, 2018 15:00:00 PST"
);

const content = "<h1>" + everydayPack.name + "</h1>";

document.body.innerHTML = content;
```
