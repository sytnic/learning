##  Создание бота @BotFather

    /newbot

Задаём  
- любое имя (например, itProger Telegram Bot)
- программное название с окончанием Bot (например, itProgerTelegramBot)

Настраиваем

    /help

    /setname 
        изменяет имя

    /setdescription 
        любое описание, появляется при первом посещении бота

    /setabouttext
        описание, которое будет постоянно у бота

    /setuserpic 
        задать своё изображение для бота
