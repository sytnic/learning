## 53. Keeping the server up to date

### Обновление сервера

Обновить список пакетов

    sudo apt-get update

Установить новые версии пакетов

    sudo apt-get upgrade

### Обновление виртуальной машины

Чтобы обновить VM -   
скачать предложенный по ссылке новый файл и установить его.  

Чтобы обновить VirtualBox Guest Additions (процесс такой же, как при установке) -  
смонтировать CD image и запустить установщик

    sudo mount /dev/cdrom /media/cdrom

    sudo sh /media/cdrom/VBoxLinuxAdditions.run --nox11

    sudo reboot
