## 012-Dialect nuances

```bash
// create collection
db.createCollection('customers')

// see inside the collection
db.customers.find();

// create statement
// insert a new document into the customers collection
// (inserting a row into a table)

db.customers.insertOne(
{
    _id : 'o23og9d039d',
    FirstName: 'Amata',
    LastName: 'Lee',
    Email: 'amata@test.com',
    Phone: '111-111-1111',
    Address: 'home',
    City: 'New York',
    State: 'New York',
    Zipcode: '10000'
}
);

// read statement, select, find statement

db.customers.find();
db.customers.find({first_name:'Amata'})

// update statement

db.customers.updateMany(
  { City: "Philadelphia" },
  {
    $set: { City : "Miami"},
  }
);

// it's case sensitive! let's try to switch it back

db.customers.updateMany(
  { City: "Miami" },
  {
    $set: { City : "Philadelphia"},
  }
);

// delete

db.customers.remove(
  {FirstName:'Amata'}
);
```
