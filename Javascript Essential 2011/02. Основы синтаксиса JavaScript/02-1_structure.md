# Структура

Javascript - это обычный текст.  
Javascript - интерпретируемый язык (не передаётся компилятору).  
Javascript чувствителен к регистру.  
Не чувствителен к пробелам.  

JAVASCRIPT COMMENTS 
```
// this is a comment  
alert("hello"); // can go here

/* this is 
 a multiple line
 comment */
```
