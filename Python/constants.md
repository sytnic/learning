## Python - Как объявить постоянную и вынести её в отдельный файл

В Python константы (переменные, значение которых нельзя изменить) обычно объявляются в отдельном файле — модуле. Это позволяет: codechick.ioguvi.in

* Организовать код — константы размещаются в одной локации, что упрощает их управление.
* Защитить пространство имён — отдельный модуль предотвращает конфликты имён.
* Обеспечить целостность данных — с помощью такого подхода можно гарантировать целостность данных во всём приложении.

 sky.pro

Объявление постоянной

Внутри модуля константы записываются заглавными буквами. Если нужно разделить слова, используются подчёркивания. Это указывает, что имя обрабатывается как константа, но на самом деле Python не гарантирует неизменяемость — программист должен оставлять значение неизменным. codechick.ioguvi.intutorialspoint.com

Пример:

* PI = 3.14

   — объявление константы PI.
  
* GRAVITY = 9.8

   — объявление константы GRAVITY.

 codechick.iopydocs.ru

Вынесение в отдельный файл

По convention модуль с константами часто называется constants.py. Пример: guvi.in

* constants.py

   — объявление констант PI и GRAVITY. codechick.ioguvi.in

* main.py

   — импорт модуля constants.py и использование констант. 

  Например:

* from constants import PI, GRAVITY

   — импорт констант из constants.py. guvi.in

* print(constant.PI)

   — вывод значения константы PI. codechick.iopydocs.ru

Важно: разделение констант помогает:

* Упростить изменение — константы можно изменять в одном месте.

* Предотвратить переназначение — использование заглавных букв не предотвращает переназначение значений констант.

## Использование модулей для констант

Хорошей практикой является определение всех констант в отдельном модуле, например constants.py:

```py
    # constants.py
    DATABASE_URL = "postgresql://user:password@localhost/db"
    MAX_USERS = 1000
    DEBUG_MODE = False
```

Затем эти константы можно импортировать в другие модули:

```py
    # В других файлах
    from constants import DATABASE_URL, MAX_USERS
    
    # Использование
    if user_count > MAX_USERS:
        print("Превышено максимальное количество пользователей")
```
